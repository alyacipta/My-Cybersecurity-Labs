# 🚩 picoCTF Challenge: Binary Digits
**Status:** Completed ✅
**Date:** April 7, 2026
**Category:** Forensics

## 📝 Deskripsi Tantangan
File ini berisi deretan angka `0` dan `1` yang terlihat seperti *noise* acak. Tugasnya adalah memulihkan informasi bermakna (Flag) dari data biner tersebut.

## 🔍 Metodologi & Logika
Dalam tantangan ini, saya mempelajari bagaimana data biner dikonversi menjadi teks yang dapat dibaca manusia:
1. **Analisis Input:** Mengidentifikasi bahwa deretan angka tersebut adalah representasi biner.
2. **Decoding:** Menggunakan teknik konversi **Binary to ASCII**.
3. **Tools:** Menggunakan **CyberChef** untuk melakukan proses *decoding* secara instan tanpa perlu menjalankan skrip rumit di terminal.

## 💡 Key Insight
Data di tingkat paling dasar komputer adalah biner. Tantangan ini memperkuat pemahaman saya tentang bagaimana karakter teks disimpan dalam memori komputer dan bagaimana cara mengekstraknya secara manual.
